Terplounge

About

Welcome to Terplounge, which enables simultaneous interpreters to practise alone, using resources supplied by professional colleagues, or using examples from real life.

Please note that this is intended as a demonstration of the capabilities of the core service, and does not have features one would expect in a finished project, such as login and security. Apologies to mobile users&em;our user base works from laptop or desktop machines, so we've prioritized those for the time being.
